# API de Consulta de Empresas (CNPJ)

Projeto desenvolvido com foco em aprendizado prático de backend utilizando FastAPI, PostgreSQL e Docker.

## 🚀 Tecnologias utilizadas

- Python
- FastAPI
- PostgreSQL
- Docker
- Uvicorn

## 📦 Funcionalidades

- Criar empresa
- Listar empresas
- Buscar empresa por CNPJ
- Atualizar empresa
- Deletar empresa
